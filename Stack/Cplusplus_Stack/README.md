# C++ Stack
In C++ Stack is a container adaptor. C++ Sequential Container Types:
<ul>
  <li>vector&emsp;r</li>
  <li>deque</li>
  <li>list</li>
  <li>forward_list</li>
  <li>array</li>
  <li>string</li>
</ul>
The stack adaptor takes an existing sequential container type (other than array or forward_list) and makes it act like a stack.
